The corrupt shard modifies an item unpredictably. You can corrupt a normal, magic, rare, or unique item. Information on what can happen can be found below.

 

When the item is a unique

50% = Nothing Happens
25% = Item downgrades to a rare , that is from the same base type as the unique, and rolls new affixes
25% = Item receives a new "Corruption Implicit" Affix that is an additional affix on the item. Now totaling 5 affixes possible. Implicit added is randomly rolled and is weighted. So the higher weight affixes roll more commonly.

 

When the item is a rare/magic/normal

50% = Nothing Happens
50% = Item receives a new "Corruption Implicit" Affix that is an additional affix on the item. Now totaling 5 affixes possible. Implicit added is randomly rolled and is weighted. So the higher weight affixes roll more commonly.

Starting Corruption Implicits - To be expanded upon later

  • +10-20 Life (500 Weighting)
  • +4-6 Defence (500 Weighting)
  • +5% Increased Attack Speed (50 Weighting)
