There are several ways you can contribute to the project. Regardless if you know how to code, sprite, or otherwise. 

All of our repos can be found here: https://github.com/orgs/Path-of-Terraria/repositories

 

Public Repositories

 

Website

https://github.com/Path-of-Terraria/website

The website requires Node 20 or later. I highly suggest using NVM to manage your node versions. This can be installed with chocolatey as well.

choco install nvm

Once you have nvm you can install Node versions. You can use nvm install 22 to get version 22 of nodejs.

This is like any other NodeJS project so you must install the dependencies. We use PNPM for this instead of NPM for various reasons. So install the package by executing pnpm i in the root directory of the project.

Once this finishes you should be able to launch the website by running pnpm run start

This will generate a URL for you to click on to see the website in real time with any of your changes.

Note that some features may fail as they require a locally running API.

 

Private Repositories

 

API

https://github.com/Path-of-Terraria/PathOfTerrariaApi

There are a few private repos due to anti-cheat methods and API authentication for the online experience. Only the Online Mod and the API itself are private and everything else is public.

To work on these repositories you will need the following installed

  • MySQL Local Server
  • MySQL Workbench (optional)

I highly suggest using Chocolatey for this

You can install these prerequisites using the following commands

choco install mysql 

choco install mysql.workbench

Once you have these you can clone the API. Once the API has been cloned you will want to Open MySQL Workbench and follow these steps to create a user and Schema

Within workbench we are going to hit the + button to add a new connection

 

We'll be connecting to our local mysql database which was installed in the chocolatey first command. For the password, the default is just blank.

Once you got all that in, should be able to test connection using the button to make sure youre all set.

If all set, click close and click the connection on the home screen to connect to that db

Should look something like

 

Perfect, now we want to click on the Schemas button

 

Then right click -> Create Schema

 

The database information when working in a .NET project is usually within your appsettings.{env}.json Here is the one for the API: https://github.com/Path-of-Terraria/PathOfTerrariaApi/blob/main/src/Api/appsettings.Development.json

"Db": "server=localhost;port=3306;database=pathofterraria;user=PotAdmin;password=<SEE REAL FILE>"

 

So your database name (schema) should be pathofterraria

 

Then click apply once you've inserted the name

 

From there we want to go to Server -> Users and Privaleges

 

Hit Add Account

 

User PotAdmin 

Password <SEE FILE>

 

From the db connection string linked above

Then click on Schema Privaleges

 

Hit Add Entry

 

Select Path of Terraria

 

Then click Select All to grant all permissions to that user

 

Then hit apply, this should create that admin user for that schema only

 

Now that we have a schema and a user, we should now be able to click Play and run the API project. All the tables and rows will be seeded automatically.
