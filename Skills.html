Skills are special abilities that can be used with hotkeys assigned in the controls settings. Skills can be unlocked through progression or item affixes.

Skills have levels that make them progressively better. Skill levels can be obtained through progression or item affixes.

Class specific skills can only be used when using a weapon that matches that of the skill.

 

Skill Specializations

Skill specializations change or modify a skill at its base layer. Each large triangle in a skill tree is a skill specialization. For example, Nova can be changed into a fire, lightning, or frost nova. Only one skill specialization can be selected for a given skill.

 

Skill Augmentations

Augmentations are itemized upgrades for skills. Augmentation orbs are in the global drop pool. Augmentation orbs can be applied to any skill to unlock an augmentation slot. These augmentation slots can then be modified to use one of the available augmentations. By default, only two augmentations can be applied to any skill, though future progression may unlock more. Skill augmentations are intended to apply to several different skills and be more generic. Some augmentations may not work for every skill specialization. For example, a fire damage augmentation would not be assignable to a skill that does not deal fire damage.
