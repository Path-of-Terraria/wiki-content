This release covers some of the important bugs from the public beta release

  • Blood Siphon only applies to Ranged weapons, properly requires a Ranged weapon
  • Added ability to restart Tutorial
  • Improved tile wand item count text, added ammo text, both only show in hotbar
  • Made hotbar items draw with the HotbarItem slot context for consistency
  • Allowed Jungle Bats to drop Bat Wings, 1/15 chance
  • Fixed issue where right clicking with the Grimoire would still try to shoot an invalid projectile
  • Fixed EoW portal despawning if EoC has been defeated, removed despawn condition entirely
  • Added clarification text to Arcane Obelisk
