Pyralis (InfernalBoss) – Balance & Mechanics Updated

This boss felt very easy for being a post moon lord boss. We wanted to truly have it be a challenge. This should bring it up to the post moon lord, stretch goals that we intended it to be.

 

Balance
  • HP: 75,000 → 225,000 (3×)
  • Contact damage: 50 → 150 (3×)
  • Fire puddle damage (SpreadTileFlames): 30 → 150 (5×)

 

Ignite Integration
  • OnFire removed: Previously this would apply the default vanilla on fire ability. This has been replaced with our in-house Ignite system

 

Ignite Synergy (new)
  • Pyralis deals +30% damage to Ignited targets
    • InfernalFlames projectile (fire puddles & burst flames)
    • InfernalBoss NPC (contact damage)

 

Heat Aura (new)

A pulsing aura mechanic

Aura radius: 336 px (~21 tiles) — ring damage
Burst radius: ~470 px (~29 tiles) — Ignite on burst

 

Drop Table Rework

  • Added normal item records back into the item drop pool: 67% normal / 30% magic / 3% rare.
  • Reduced unique item weighting globally by 5x.
  • Reduced automatic rarity scaling from item level so late-game drops don’t skew so hard into rare/unique.
  • Scaled overall mob drop quantity from about 0.5x early to 0.2x late.
  • Added map tier quantity scaling: higher map tiers now give more drops.
  • Fixed map affix rarity boost math/display so affix strength applies sanely.
  • Fixed player item drop rate multiplier so it affects the drop count before rolling.
  • Fixed fractional drop-count logic so partial drops behave like real percentages.
  • Currency/shard category chance now scales from 0.5x early to 0.2x late
  • Fixed a small shared-modifier issue where gear and currency pool construction reused the same mutated rarity modifier.
  • Unfolding Shards had their weight reduced by half
  • Glimmering Shards had their weight reduced by half
  • Increase chance of Shifting Shards
  • Reduced base PoT related drop rates for everything by about 40%

 

Chest Loot Improvements

Map chests have been significantly improved and should now feel much more rewarding to open:

  • Map Rarity now effect chest loot (previously was a bug that prevented this)
  • More loot Chests now contain up to 5 pieces of gear/currency (up from 3).
  • Better rarity All map chests have a built-in +50% rarity boost, meaning Rare and Unique items are more likely toappear. This bonus stacks with any rarity affixes on your map.
  • Smart Loot Gear found in chests now favors higher affix tiers, similar to how curated loot sources work in other ARPGs. You're less likely to find a max-level sword with only the weakest affixes.
