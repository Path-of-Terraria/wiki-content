Shrines are a mapping mechanic that can appear inside explorable maps.

 

According to the main patch notes, shrines can spawn by default in explorable maps, spawn a pack of enemies around themselves, and can be right-clicked to grant a temporary buff.

 

How Shrines Work

  • Shrines can appear naturally in explorable maps.
  • Each shrine spawns enemies around it.
  • Enemies in the shrine's area are enhanced by that shrine's effect.
  • Right-clicking a shrine grants the player that shrine's buff for a duration.

 

Known Shrine Types

  • Godlike
  • Hardiness / Defense
  • Restoration
  • Unstoppable
  • Haste

 

The original patch notes use the name Hardiness, while extracted shrine item and projectile names use Defense.

 

Theme Variants

Extracted item data also shows biome-specific shrine variants for at least Forest and Desert, which suggests shrine visuals or placements can differ by map theme.

 

Important Note

These mapping shrines are separate from decorative structures such as the Glade Shrine shown on the Forest Map page. The Glade Shrine is a point of interest, while the shrine mechanic described here is a combat-and-buff system.
