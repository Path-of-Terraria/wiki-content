World's End is Azariel's endgame quest and currently covers both the Lunatic Cultist and Moon Lord domains.

 

Prerequisites

  • Holy Absolution must be completed.
  • Empress of Light must already be defeated.

 

Steps

  1. Return to the Overworld and prepare yourself.
  2. Talk to Azariel to continue the quest.
  3. A portal is opened at Ravencrest's chamber.
  4. Enter the Lunatic Cultist's domain.
  5. Defeat Lunatic Cultist.
  6. Enter the Moon Lord's domain.
  7. Defeat Moon Lord.
  8. Return to Azariel to complete the quest.

 

Rewards

  • 100,000 experience

 

Notes

  • The localization mentions biome sigils, but the current quest code does not yet use that collection step.
  • The source code also contains TODOs for post-Cultist pillar progression and other endgame handoff steps.
