0.1.10

  • Finishes the Morgana NPC Sprite sheet. This should resolve the issues where Morgana's sprite would change while she was walking
  • Fixes the Echoing, Mystic, and Shifting Shard Sprite Sheets. They should no longer float weird in their animations
  • Fixes unique items that explicitly set their name to a color (such as burning red boots setting it's name to red) not looking correct in the tooltip
  • Fixes the inability to equip Path of Terraria gear
  • Adds in health and mana portion drops to normal mobs (1/7 Chance)
  • Fixes an exploit that allowed weapon swapping before the usage of the item was complete. IE no longer able to warshield spam.
  • Fixes closing the passive tree with the Escape key. Should properly toggle now.
  • Fixes an issue where Garrick would sometimes de-spawn while completing the King Slime quest. Halting progress.
  • Fixes an issue where the Goblin Scout would spawn prematurely in Ravencrest
  • Fixes an issue where Ravencrest would sometimes spawn in Hardmode before completing hardmode on the overworld.
